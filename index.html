Simple Documentation

A simple set of examples to illustrate some functions of the calculator:

  • All angles are accepted in Radians. To find cosine of 45, for example, type cos(45)
  • Same formatting is required for square root function. As an illustration: type sqrt(4), to find square root of 4.
  • The calculator can evaluate expressions where brackets are used, without multiplication symbols. e.g: 2 (2) (3)=12
  • To reset, use either CA, or AC buttons. CE deletes the last character that was typed.
